# Tarea Desafío 1 UDB-HTML

## Criterios a evaluar

> 
>* Diseño responsive con Bootstrap.

>* Aplicar el uso de Bootstrap como librería para la creación de diseños adaptables a las diferentes resoluciones (Responsive Web Design) y realizar una página web del tema que usted prefiera, que contenga la implementación de los siguientes literales:
>* Diseñando con rejillas de Bootstrap. Manejo de filas y columnas. (25%)
>* Tipos de contenedores. Imágenes responsivas. (25%)
>* Tablas responsive. (25%)
>* Formularios responsivos. (25%)


>


## Levantar proyecto
Tener instalado previamente Git y cuenta creada en Github luego clonar el repositorio utilizando el siguiente comando:
```
git clone https://github.com/DavidGalileo24/desafio1_udb.git
```
Posteriormente abrir una consola dentro de la carpeta del proyecto e instalar las dependencias del proyecto donde traera la instalación de Bootstrap y otras que se lleguen a utlizar corriendo el siguiente comando:
```
npm install
```


Instalar la extensión de `Live Server (Five)` en el VSCode y en el index.html que esta afuera en raíz dar clic derecho abrir con Live server para que pueda mostrarse el proyecto.
